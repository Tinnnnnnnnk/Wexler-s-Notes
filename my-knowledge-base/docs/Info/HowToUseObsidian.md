## 笔记中的链接是纯文本
- 假如我写了一段链接在笔记中，但是我不需要去访问这个链接，我只是想将其作为一个纯文本，那么就需要给这个链接加上一个反引号，例如 `www.baidubaidu.com`
- 如果不加这个反引号，那么**VitePress**就无法将这个**笔记文件**打包

## 操作细节

| **你想做的**  | **你要输入的符号**   |
| --------- | ------------- |
| **变大标题**  | `#` 或 `##`    |
| **变圆点列表** | `-` (减号加空格)   |
| **变数字列表** | `1.` (数字点空格)  |
| **变引用**   | `>` (大于号空格)   |
| **变代码**   | `` ` `` (反引号) |
| **画横线**   | `---`         |
