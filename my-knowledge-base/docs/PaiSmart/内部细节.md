## 2.文档处理模块
### 1.1Apache Tika
- 是 Java 世界里的“万能文档解析器”，能够从 **1000 多种** 不同的文件格式（如 PDF, Word, Excel, PPT, HTML, XML, 图片等）中检测并提取
#### A 格式检测 (MIME Type Detection)
- Tika 不仅仅看文件后缀名（因为后缀名可以随便改，比如把 `.exe` 改成 `.jpg`）。 Tika 会读取文件的**前几个字节（Magic Bytes）**。每种文件格式都有独特的“指纹”，Tika 通过指纹来判断这到底是个什么文件。
#### B 内容提取 (Content Extraction)

这是 RAG 最需要的功能。Tika 内部集成了大量的第三方解析库：

- 解析 PDF，它用 **Apache PDFBox**。
    
- 解析 Word/Excel/PPT，它用 **Apache POI**。
    
- 解析 HTML，它用 **TagSoup**。
    
- 解析 图片 OCR，它集成 **Tesseract**。
    

**Tika 就是这些库的“总指挥”。**
#### C. 元数据提取 (Metadata Extraction)

有时候你需要知道这篇文档是谁写的，或者什么时候写的（用于过滤过期的文档）。Tika 可以把这些信息提取成 Key-Value 对。

#### D 代码示例
 ```java
// 引入依赖
// org.apache.tika:tika-core
// org.apache.tika:tika-parsers

import org.apache.tika.Tika;
import java.io.File;

public class DocumentParser {
    public String parse(File file) {
        Tika tika = new Tika();
        try {
            // 自动检测类型并提取文本
            String content = tika.parseToString(file);
            return content;
        } catch (Exception e) {
            e.printStackTrace();
            return "";
        }
    }
}
 ```
