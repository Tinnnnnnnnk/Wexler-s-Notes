### Java有哪些数据类型  
#### a.基本数据类型
- 字符型
	- char
- 数值型
	- 整型
		- long
		- int
		- short
	- 浮点型
		- float
		- double
	- 字节型
		- byte
- 布尔型
	- boolean

| 数据类型    | 默认值      | 大小      |
| ------- | -------- | ------- |
| boolean | false    | 1字节或4字节 |
| char    | '\u0000' | 2字节     |
| byte    | 0        | 1字节     |
| short   | 0        | 2字节     |
| int     | 0        | 4字节     |
| long    | 0L       | 8字节     |
| float   | 0.0f     | 4字节     |
| double  | 0.0      | 8字节     |

#### b.引用数据类型
- 接口 (interface)
- 类 (class)
- 数组
