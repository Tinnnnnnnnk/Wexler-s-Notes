# Wexler Notes 网站修改指南

本文档说明如何修改这个基于 VitePress 构建的知识库网站。

---

## 项目概述

这是一个使用 **VitePress** 构建的个人知识库网站，采用独特的"油画冷色调"主题设计。

---

## 文件结构总览

```
my-knowledge-base/
├── docs/
│   ├── index.md                          # 首页内容
```

## 一、修改网站内容（文章/页面）

**目录**: `docs/`

这是所有 Markdown 内容文件存放的位置。每个子目录对应一个知识模块：

| 目录 | 内容说明 |
|------|---------|
| `Code/` | 算法与数据结构 |
| `Sky-Take-Out/` | 项目实战 |
| `PromptLearning/` | 提示词工程 |
| `PaiSmart/` | 过程复盘 |
| `Resume/` | 简历 |
| `Info/` | 个人信息 |

**修改方式**: 直接编辑对应的 `.md` 文件即可。

---

## 二、修改首页内容与布局

**文件**: `docs/index.md`

这是网站的主页，包含以下区块：

- **Hero 区域**: 标题、副标题、标语、按钮
- **功能介绍卡片**: BUILD LOOP、LEARNING SYSTEM、DELIVERY STYLE
- **系统状态卡片**: Deploy Pipeline、Algorithm System、Interview Assets
- **路线图**: 从输入到交付的主路径
- **知识模块卡片**: 各个主要模块的入口卡片

## 三、修改样式与主题风格

**主要样式文件**: `docs/.vitepress/theme/style.css`

### 颜色系统

颜色变量定义在 `:root`（浅色模式）和 `.dark`（深色模式）中：

| 变量名 | 浅色模式 | 深色模式 | 用途 |
|--------|---------|---------|------|
| `--oil-bg` | `#d8dee6` | `#0f1620` | 主背景色 |
| `--oil-bg-deep` | `#ccd4de` | `#0a1119` | 深背景色 |
| `--oil-surface` | `#edf1f6` | `#172332` | 表面色 |
| `--oil-ink` | `#1f2a38` | `#e6edf5` | 主文字色 |
| `--oil-muted` | `#556272` | `#b2c0cf` | 次要文字色 |
| `--oil-accent` | `#4a6d86` | `#88a8c0` | 强调色 |

### 背景效果

```css
/* 背景渐变 - 径向渐变叠加 */
background:
  radial-gradient(circle at 10% -8%, var(--oil-bg-gradient-1), transparent 32%),
  radial-gradient(circle at 88% 16%, var(--oil-bg-gradient-2), transparent 30%),
  radial-gradient(circle at 72% 86%, var(--oil-bg-gradient-3), transparent 30%),
  linear-gradient(180deg, var(--oil-bg) 0%, var(--oil-bg-deep) 100%);
```

### 字体设置

```css
--vp-font-family-base: "Avenir Next", "Segoe UI", "PingFang SC",
  "Hiragino Sans GB", "Noto Sans SC", "Microsoft YaHei", sans-serif;
--vp-font-family-mono: "JetBrains Mono", "Cascadia Code", "Fira Code",
  "Consolas", monospace;
```

## 四、修改网站配置

**文件**: `docs/.vitepress/config.mjs`

### 可配置项

| 配置项 | 位置 | 说明 |
|--------|------|------|
| 网站标题 | `title` | 浏览器标签和 SEO 标题 |
| 网站描述 | `description` | SEO 描述信息 |
| Logo | `themeConfig.logo` | 导航栏 Logo 图片路径 |
| 导航栏 | `themeConfig.nav` | 顶部导航链接 |
| 社交链接 | `themeConfig.socialLinks` | GitHub 等社交媒体链接 |
| 页脚信息 | `themeConfig.footer` | 页面底部信息 |
| 编辑链接 | `themeConfig.editLink` | 编辑此页的链接 |

### 示例：修改导航栏

```javascript
nav: [
  { text: '🏠 首页', link: '/' },
  { text: '📚 核心知识库', link: '/Sky-Take-Out/00-后端开发知识大本营' },
  { text: '🎨 风格实验室', link: '/Style-Lab/00-风格入口' },
],
```

## 五、修改自定义组件

这些 Vue 组件位于 `docs/.vitepress/theme/components/`

### HomeFxBackdrop.vue - 首页背景效果

功能：
- 全站液态/晶透背景层
- 首页视频/图片背景
- 液态气泡动画效果
- 首页音乐播放器

关键配置：
```javascript
const IMAGE_SRC = ''                    // 自定义背景图片
const VIDEO_SRC = '/media/home-bg/Bg1.mp4'  // 背景视频
const LIQUID_BGM_SRC = '/media/home-bgm/liquid-bgm.flac'  // 背景音乐
```

### HomeFxToggle.vue - 主题切换按钮

功能：
- 三种视觉模式切换：常态(default)、晶透(glass)、液态(liquid)
- 主题切换动画效果

### ReadingEnhancer.vue - 阅读增强功能

功能：
- 阅读进度条
- 章节导航面板（可拖拽定位）
- 回到顶部按钮

## 七、修改主题入口

**文件**: `docs/.vitepress/theme/index.js`

```javascript
import DefaultTheme from 'vitepress/theme'
import { h } from 'vue'
import ReadingEnhancer from './components/ReadingEnhancer.vue'
import HomeFxBackdrop from './components/HomeFxBackdrop.vue'
import HomeFxToggle from './components/HomeFxToggle.vue'
import './style.css'

export default {
  extends: DefaultTheme,
  Layout: () => {
    return h(DefaultTheme.Layout, null, {
      'layout-top': () => [h(HomeFxBackdrop), h(ReadingEnhancer)],
      'nav-bar-content-after': () => h(HomeFxToggle)
    })
  }
}
```

## 快速修改清单

| 修改需求 | 目标文件 |
|---------|---------|
| 改网站标题/描述 | `config.mjs` |
| 改导航栏链接 | `config.mjs` 中的 `nav` |
| 改首页文字内容 | `index.md` |
| 改颜色主题 | `style.css` 开头的变量 |
| 改背景视频 | 替换 `public/media/home-bg/Bg1.mp4` |
| 改背景音乐 | 替换 `public/media/home-bgm/liquid-bgm.flac` |
| 改 Logo | 替换 `public/images/logo.jpg` |
| 改文章内容 | 对应目录下的 `.md` 文件 |
